bOs - Minimal Batch Operating System
====================================

DESCRIPTION:
bOs is a lightweight operating system environment built entirely with
Windows Batch scripting. It provides a simulated OS experience with
core system functions, applications, and games.

FEATURES:
- Command-line interface with prompt
- Modular kernel system
- File manager
- System information viewer
- Built-in applications (calculator, clock, editor)
- Mini-games
- Configuration system
- User management

INSTALLATION:
Simply extract to any folder and run BOOT.BAT

SYSTEM REQUIREMENTS:
- Windows XP or newer
- Command Prompt access
- 1MB free space

COMMANDS:
Type 'help' in bOs for available commands

FOLDER STRUCTURE:
/bOs
  ├── BOOT.BAT          - Main bootloader
  ├── core/             - Kernel files
  ├── modules/          - System modules
  ├── system/           - System utilities
  ├── apps/             - Applications
  ├── config/           - Configuration
  ├── storage/          - User data
  ├── users/            - User profiles
  └── logs/             - System logs

DEVELOPMENT:
Add your own modules in /modules
Add applications in /apps
Modify system settings in /config
